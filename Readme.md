# Calendar meetings

## What is the proejct

This is a Java project which enables the user to find possible meeting times for 2 employees. Using the input of 2 employees' meetings, their start and end of availability and desired meeting duration. 

The current iteration of the project is backend only and contains no UI. 

## Potential future improvements

- Add frontend capability for users to add meetings

- Add frontend capability for users to read availability for meetings with multiple employees

- Addition of Employee entity to increase the scope of additional features

- Enable the application to handle more than 2 employees' meetings
